## MyProt_AI 是一种报文协议工具

- 这是一种报文协议工具，基于socket，现demo版支持ModBus协议。

- model层有设备、标签、协议。

- 协议配置名称，通讯方式，数据帧，超时时间，握手协议，报文。

- 握手协议，报文：名称，报文模板（固定值，模板：参数 + 类型 | 名称 + 函数 + 类型），返回报文解析。

- 设备配置名称，协议类型，ip，端口。

- 标签配置名称，设备名称，协议名称，参数。

- 本组件终身开源免费，采用最宽松MIT协议，您也可以随意修改和商业使用（商业使用请做好评估和测试）。  

- 开发工具：Visual Studio 2019 


## 逻辑

- 基于16位字节数组发送信息，配置发送消息的参数。

- 将返回的报文解析。
